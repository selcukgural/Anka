## Reproducing

```bash
# 1. Start PostgreSQL and create schema:
psql -h localhost -U benchmarkdbuser -d hello_world -f Test/LoadTest/Anka.Wrk.LoadTest/scripts/setup_db.sql

# 2. Run the benchmark (DATABASE_URL defaults to localhost if unset):
dotnet run --project Test/LoadTest/Anka.Wrk.LoadTest --configuration Release
```

Dependencies: [wrk](https://github.com/wg/wrk) must be on `PATH`.
On macOS: `brew install wrk`. On Ubuntu: `apt install wrk`.
